# Poem Generator

Generate random poems from your own words with customizable structure.

## Features
- Custom stanza count
- Custom line count
- Custom words per line
- Random connectors and punctuation
- Saves output to `poem.txt`

## Usage
1. Run the script
2. Enter your words separated by commas
3. Set stanza count, lines per stanza, and words per line
4. Get your poem

## Run
```bash
python main.py
